Excel Format Instructions

The Excel file should contain the following sheets:

Contains basic user information and notification settings:

  • unique_userid: Unique identifier for each user
  • user_id: User's login ID
  • user_name: User's display name
  • location: User's location (Required)
  • activation_status: 1 for active, 0 for inactive
  • expiry_date: Subscription expiry (YYYY-MM-DD)
  • fixed_lat: Latitude coordinate
  • fixed_lon: Longitude coordinate
  • password: User's password
  • mode_only_preferred: 1 to enable
  • non_good_deals: 1 to enable
  • good_deals: 1 to enable
  • near_good_deals: 1 to enable

Contains product preferences for each user:

  • unique_userid: Must match a user in the Users sheet
  • name: Product name (e.g., iPhone 16 Pro)
  • min_price: Minimum acceptable price
  • max_price: Maximum price willing to pay
  • preferred: 1 for yes, 0 for no

These additional sheets all follow a similar format:

  • Keywords Sheet: unique_userid, keyword
  • Excluded Words Sheet: unique_userid, excluded_word
  • Resellers Sheet: unique_userid, reseller_name

Each row represents one item for one user, and the unique_userid must match a user in the Users sheet.

Tip: The easiest way to get started is to first export your data, which will create a properly formatted Excel file with all the required sheets.
