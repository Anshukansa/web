CSV Format Instructions

The CSV file should have the following columns:

Column Name Description Format Example
unique_userid Unique identifier for each user Text user_123
user_id User's ID Text john_doe
user_name User's name Text John Doe
location User's location Text Los Angeles
activation_status Activation status 1 (active) or 0 (inactive) 1
expiry_date User's subscription expiry date YYYY-MM-DD 2025-12-31
fixed_lat Latitude coordinate Decimal number 34.0522
fixed_lon Longitude coordinate Decimal number -118.2437
password User's password Text password123
products Product preferences name:min_price:max_price:preferred; (semicolon-separated) iPhone 15 Pro:0:800:1;iPhone 14:0:500:0;
keywords Search keywords Semicolon-separated list mint;unlocked;new;
excluded_words Words to exclude from search Semicolon-separated list broken;cracked;damaged;
resellers Preferred resellers Semicolon-separated list Apple;Best Buy;Walmart;
mode_only_preferred Only send preferred models 1 (enabled) or 0 (disabled) 1
non_good_deals Send all listings 1 (enabled) or 0 (disabled) 0
good_deals Only send good deals (below max price) 1 (enabled) or 0 (disabled) 0
near_good_deals Send deals near max price 1 (enabled) or 0 (disabled) 1
Note: You can download a template by exporting your current data and modifying it.
